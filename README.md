# MakeTS

MakeTSは、任意の入力映像を、簡易的な番組情報などとともにMPEG-2 TS形式へエンコードするツールです。
架空放送局の作品制作などにご利用いただけます。

## 使い方

お好きな映像を読み込み、解像度、副音声、チャンネル情報、番組情報などを必要に応じて指定して出力すると、対応ソフトで再生可能なTSファイルを生成できます。

## 導入方法

ダウンロードした圧縮ファイルを解凍し、任意の場所に配置したうえで、exe_filesフォルダにffmpeg.exe、ffprobe.exe、tsp.exe (TSDuck) を配置していただければ、すぐにご利用いただけます。
念のため、日本語といった英字でないものを含むディレクトリには配置なさらないでください。

## 必要環境

- ffmpeg
- TSDuck

映像・音声のエンコードにはffmpeg、番組情報の付加にはTSDuckがそれぞれ必要です。お手数ですが、必要に応じてご自身でインストールなさってください。

## 制約

現在、以下の機能には対応しておりません。
- ロゴデータの添付
- データ放送の添付
- ワンセグ・サブチャンネル映像の指定
- デュアルモノ音声の指定
- 後続番組情報の追加
- HDR向けなどの色空間変換への対応
