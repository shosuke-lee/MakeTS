# MakeTS

MakeTSは、任意の入力映像を、簡易的な番組情報などとともにMPEG-2 TS形式へエンコードするツールです。
架空放送局の作品制作などにご利用いただけます。

## 使い方

お好きな映像を読み込み、解像度、副音声、チャンネル情報、番組情報などを必要に応じて指定して出力すると、対応ソフトで再生可能なTSファイルを生成できます。

## 導入方法

ダウンロードした圧縮ファイルを解凍し、任意の場所に配置すれば、すぐにご利用いただけます。
念のため、日本語といった英字でないものを含むディレクトリへの配置は避けてください。

## 対応環境

- Windows 11
その他の環境については、今後検証を進める予定です。

## 対応言語

- 日本語
- English
近いうちに中国語への対応も予定しています。

## 必要環境

- ffmpeg
- TSDuck
映像・音声のエンコードにはffmpeg、番組情報の付加にはTSDuckがそれぞれ必要ですから、お手数ですが必要に応じてご自身でインストールなさってください。

## 制約

現在、以下の機能には対応していません。
- ロゴデータの添付
- データ放送の添付
- ワンセグ・サブチャンネル映像の指定
- デュアルモノ音声の指定
- HDR向けなどの色空間変換への対応
